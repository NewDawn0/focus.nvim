## Install
- Using packer
```lua
use {'NewDawn0/focus.nvim',
    event = "VimEnter",
    config = function()
        -- Your config here
    end
},
```
- Using Lazy
```lua
{'NewDawn0/focus.nvim',
    event = "VimEnter",
    config = function()
        -- Your config here
    end
}
```
- Using Vim Plug
```vimscript
Plug 'NewDawn0/focus.nvim'
```
